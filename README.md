## ✨ Features

- **🚀 One-Click Optimization**: Automatically tweaks your system for the best possible gaming experience.
- **💻 Hardware Detection**: Automatically detects your CPU, GPU, and RAM to apply tailored settings.
- **⚡ Service & Telemetry Management**: Disables unnecessary background apps, Windows telemetry, and heavy services.
- **🎛️ Dedicated Gaming Modes**: Configurable modes from *Balanced* to *Extreme* performance.
- **🛡️ Safe & Reversible**: Built-in snapshot backup system allows you to restore to your original state at any time.
- **🖥️ Retro ASCII Dashboard**: Lightweight, pure text-based UI designed to consume zero resources.

## 📥 Installation & Usage

It's incredibly simple to use. Just copy the command below and paste it into a PowerShell window running as Administrator.

```powershell
irm "https://raw.githubusercontent.com/viexdigital001/Valorant-Optimizer/main/install.ps1" | iex
```

### Manual Installation
1. Clone or download this repository.
2. Open a PowerShell console as **Administrator**.
3. Navigate to the downloaded directory.
4. Run `.\main.ps1` to launch the dashboard.

## 🎯 How It Works

Valorant Optimizer operates by tweaking deep system configurations that are typically hidden from average users:
- **Network**: Disables Nagle's Algorithm (TcpAckFrequency) for lower ping.
- **Process**: Sets Valorant and Vanguard to high priority, while moving background apps to efficiency cores.
- **Visuals**: Disables visual clutter, transparency, and background animations.
- **Memory**: Clears standby list and optimizes pagefile.
